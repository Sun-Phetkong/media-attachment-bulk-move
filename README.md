# Media Attachment Bulk Move

A WordPress plugin to view and bulk manage media files attached to posts and pages, directly from the post editor.

## What it does

- **View all attachments** — Adds an "Attached Media Files" meta box to all public post types, showing every media file attached to that post in a grid layout.
- **Bulk select & move** — Select multiple files with checkboxes and move them to another post/page using an autocomplete search.
- **File preview** — Thumbnail previews for images; mime-type icons for other file types, along with file size and upload date.
- **Quick actions** — View, edit, detach (unlink from post without deleting), or permanently delete individual files.
- **Works with all post types** — Automatically available on every public post type.

## Requirements

- WordPress 5.0+
- PHP 7.4+

## Installation

1. Upload the `media-attachment-bulk-move` folder to `/wp-content/plugins/`
2. Activate via the Plugins menu
3. Open any post or page editor to see the **Attached Media Files** meta box
